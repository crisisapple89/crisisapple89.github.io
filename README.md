# What is this?
This a website I made that allows you to play games in school without your schools content filter blocking everything.

# Features:
- Adrift (Proxy)
- INSECURLY
- WebRetro emulator
- about:blank embedder
- Suggestions form
- Alot of games! (More will be added over time)
